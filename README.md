# Blockchain Project

## Technologies Used:
- Next.js for web application
- Hardhat for Ethereum network

### How to run:
- run 
```shell
npm install
``` 
in the root folder of the project to download dependencies
- run
```shell
npx hardhat node
```
to run the hardhat network
- run
```shell
npm run dev
```
to run the web app

other commands:
```shell
npx hardhat run scripts/deploy.js
```
